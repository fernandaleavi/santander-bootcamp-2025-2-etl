# Santander Bootcamp Ciência de Dados 2025-2

## Sobre o projeto

Esse repositório foi criado para o desafio de projeto 'Explorando IA Generativa em um Pipeline de ETL com Python' do bootcamp 'Santander Bootcamp 2025-2 - Ciência de Dados com Python'. Durante a execução do projeto, foi constatado que o repositório fornecido pela DIO para resolução do desafio está fora do ar; portanto, foi usada uma outra API desenvolvida especialmente para este desafio, a [Fake API - Santander (DIO)](https://github.com/PauloHLeme/Fake-API-Santander-Dio/tree/main). 

## Objetivo

Criar uma pipeline de ETL usando API de IA generativa, para criar novas mensagens personalizadas sobre investimento para cada cliente do banco.
